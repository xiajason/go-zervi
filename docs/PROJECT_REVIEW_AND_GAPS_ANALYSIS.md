## 📋 一、项目概况

### 1.1 项目定位

**Zervigo (JobFirst)** 是一个融合 **AI智能匹配** 和 **区块链审计** 技术的**新一代智能招聘SaaS平台**。

**核心特点**：
- 🏗️ 基于自研 **Go-Zervi** 微服务框架
- 🤖 集成 AI 智能简历分析和职位匹配
- 🔐 区块链数据审计和不可篡改记录
- 📊 三层微服务架构（Core/Business/Infrastructure）

### 1.2 目标用户

- **B端企业**: 中小型企业、猎头公司、人力资源外包公司
- **C端求职者**: 应届生、跳槽者、自由职业者
- **管理员**: 平台运营和管理人员

## 🎯 二、业务场景与功能价值

### 2.1 核心业务流程

```
【求职者】
注册登录 → 创建简历 → AI智能分析 → 搜索职位 → AI匹配推荐 → 查看匹配度 → 投递申请

【企业HR】
企业注册 → 企业认证 → 发布职位 → AI推荐候选人 → 查看简历匹配度 → 筛选面试

【区块链审计】
简历修改 → 权限变更 → 状态变化 → 全部上链记录 → 数据不可篡改 → 审计追踪
```

### 2.3 技术创新点

#### **Go-Zervi 框架** 🚀
```
创新理念 = Go-Zero优秀设计 + 自主创新架构
```

**核心优势**：
- ✅ 三层微服务架构（清晰分层）
- ✅ 统一模块命名规范
- ✅ 共享库统一管理
- ✅ 复杂技术栈支持（PostgreSQL + Redis + Consul）
- ✅ 中央大脑智能路由

#### **AI 智能服务** 🤖
- **智能匹配**: 6维度匹配算法（技能、经验、教育、地点、薪资、文化）
- **简历分析**: 技能评分、经验分析、职业发展建议
- **AI聊天**: 职业咨询、简历指导
- **智能推荐**: 个性化职位/候选人推荐

#### **区块链审计** 🔐
- **不可篡改**: 关键操作上链存储
- **数据验证**: 区块链与数据库一致性校验
- **审计追踪**: 完整操作历史记录
