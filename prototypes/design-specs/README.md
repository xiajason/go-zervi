# 设计规范文档

## 颜色规范

### 主色调
- 主色：#1976D2（蓝色）
- 辅助色：#26A69A（青色）
- 强调色：#FF5252（红色）

### 中性色
- 文本主色：#212121
- 文本次色：#757575
- 分割线：#E0E0E0
- 背景色：#F5F5F5

## 字体规范

### 字体大小
- 标题：36rpx / 18px
- 副标题：32rpx / 16px
- 正文：28rpx / 14px
- 辅助文字：24rpx / 12px

### 字重
- 加粗：600
- 常规：400
- 细体：300

## 间距规范

基于 4px 基准网格：
- 4rpx / 2px
- 8rpx / 4px
- 16rpx / 8px
- 24rpx / 12px
- 32rpx / 16px
- 48rpx / 24px

## 圆角规范

- 小圆角：8rpx / 4px
- 中圆角：16rpx / 8px
- 大圆角：32rpx / 16px

## 图标规范

- 尺寸：48rpx x 48rpx / 24px x 24px
- 格式：SVG / PNG（2x/3x）
- 颜色：支持主题色切换
