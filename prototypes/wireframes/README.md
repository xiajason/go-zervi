# 线框图

## 页面结构

### 首页
- 顶部导航栏
- Banner 轮播
- 通知消息
- 快捷入口（4 宫格）
- 职位推荐列表

### 简历
- 简历列表
- 简历编辑表单
- 简历预览
- 权限设置

### 职位
- 职位列表（列表/网格切换）
- 职位搜索/筛选
- 职位详情页

### 个人中心
- 用户信息
- 功能入口
- 设置
- 关于

## 交互流程

### 登录流程
1. 输入手机号/密码
2. 验证码校验
3. 登录成功跳转首页

### 简历创建流程
1. 选择模板
2. 填写信息
3. 预览确认
4. 发布
